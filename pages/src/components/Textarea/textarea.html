Textarea



HTML


    <textarea class="content1" placeholder="default content"></textarea>
    <button>get content length</button>

JavaScript


    $('textarea').bizTextarea({
        theme: 'gray'
    });
    $('button').bizButton({
        theme: 'blue-gray'
    }).click(function() {
        bizui.alert($('.content1').bizTextarea('length'));
    });

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[disabled] Boolean 禁用，默认 false
[theme] String 主题

Methods

destroy()

销毁

disable()

禁用

enable()

激活

length()

获取文本长度（不计回车）
