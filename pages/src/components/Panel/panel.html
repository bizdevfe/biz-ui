Panel

Content

HTML


    <div class="panel-content" data-title="Title">Content</div>
    <button id="b5">open panel</button>

JavaScript


    $('button').bizButton({
        theme: 'blue-gray'
    }).click(function() {
        $('.panel-content').bizPanel('open');
    });
    $('.panel-content').bizPanel({
        buttons: [
            {
                text: 'OK',
                icon: 'done',
                onClick: function() {
                    this.close();
                }
            },
            {
                text: 'Cancel',
                icon: 'close',
                theme: 'gray',
                onClick: function() {
                    this.close();
                }
            }
        ]
    });

Options

参数名称 参数类型 描述
[buttons] Array 底部按钮的 option 数组（多一个 onClick 属性），默认 [ ]
[customClass] String 自定义 class
[destroyOnClose] Boolean 关闭时销毁对话框，默认 false
[marginLeft] Number 左边距（px），默认 200
[speed] Number 动画速度（ms），默认 300
[theme] String 主题
[title] String 对话框标题，默认取目标元素的 data-title 属性
[onBeforeClose] Function 关闭前回调，返回 false 则不关闭
[zIndex] Number 对话框 z-index

Methods

close()

关闭对话框

destroy()

销毁

open()

打开对话框

title([String title])

获取/设置标题

updateButtons(Object buttonOption)

更新按钮
