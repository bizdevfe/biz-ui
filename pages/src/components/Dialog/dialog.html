Dialog

I'm fixed.
I'm absolute.
I'm draggable.

HTML


    <div class="dialog-content d1" data-title="Fixed">I'm fixed.</div>
    <div class="dialog-content d2" data-title="Absolute">I'm absolute.</div>
    <div class="dialog-content d3" data-title="Draggable">I'm draggable.</div>
    <button id="b1">Fixed dialog</button>
    <button id="b2">Absolute dialog</button>
    <button id="b3">Draggable dialog</button>
    <button id="b4">Alert dialog</button>
    <button id="b5">Confirm dialog</button>

JavaScript


    $('button').bizButton({
        theme: 'blue-gray'
    });
    $('.d1').bizDialog({
        buttons: [
            {
                text: 'OK',
                icon: 'done',
                onClick: function() {
                    this.close();
                },
                theme: 'teal'
            }
        ],
        theme: 'teal'
    });
    $('.b1').click(function() {
        $('.d1').bizDialog('open');
    });
    $('.d2').bizDialog({
        buttons: [
            {
                text: 'OK',
                icon: 'done',
                onClick: function() {
                    this.close();
                },
                theme: 'teal'
            }
        ],
        position: 'absolute',
        theme: 'teal'
    });
    $('.b2').click(function() {
        $('.d2').bizDialog('open');
    });
    $('.d3').bizDialog({
        buttons: [
            {
                text: 'OK',
                icon: 'done',
                onClick: function() {
                    this.close();
                },
                theme: 'teal'
            }
        ],
        draggable: true,
        theme: 'teal'
    });
    $('.b3').click(function() {
        $('.d3').bizDialog('open');
    });
    $('.b4').click(function() {
        bizui.alert({
            title: 'Alert',
            content: 'Content',
            theme: 'red'
        });
    });
    $('.b5').click(function() {
        bizui.confirm({
            title: 'Confirm',
            content: 'Content',
            theme: 'green'
        });
    });

Options

参数名称 参数类型 描述
[buttons] Array 底部按钮的 option 数组（多一个 onClick 属性），默认 [ ]
[customClass] String 自定义 class
[destroyOnClose] Boolean 关闭时销毁对话框，默认 false
[draggable] Boolean 可拖拽，默认 false
[height] Number 对话框高度，默认取填充内容后的高度，最小 150px
[position] String 定位方式（fixed | absolute），默认 'fixed'
[theme] String 主题
[title] String 对话框标题，默认取目标元素的 data-title 属性
[width] Number 对话框宽度，最小 320px
[onBeforeClose] Function 关闭前回调，返回 false 则不关闭
[zIndex] Number 对话框 z-index

Methods

close()

关闭对话框

destroy()

销毁

open()

打开对话框

title([String title])

获取/设置标题

updateButtons(Object buttonOption)

更新按钮

bizui.alert([Object|String options])

提示对话框

参数名称 参数类型 描述
[content] String 内容，默认 ''
[okText] String 确认文字，默认 '确定'
[theme] String 主题
[title] String 标题

bizui.confirm([Object options])

确认对话框

参数名称 参数类型 描述
[content] String 内容，默认 ''
[cancelText] String 取消文字，默认 '取消'
[okText] String 确认文字，默认 '确定'
[theme] String 主题
[title] String 标题
[onOK] Function 确认回调，返回 false 则不关闭
