HTML


    <input type="text" class="date" value="2046-01-01">
    <span class="icon-date"><i class="biz-icon">&#xe916;</i></span>

    <!-- range 类型，class 必须含 input-daterange -->
    <div class="input-daterange">
        <input type="text" class="start" value="2016-01-01">
        <span class="icon-range">to</span>
        <input type="text" class="end" value="2017-12-31">
    </div>

    <!-- inline 类型 -->
    <div class="inline-date" data-date="2017-01-01"></div>
    <span class="selected-date"></span>

JavaScript


    $('input').bizInput();
    $('.date').bizCalendar({
        daysOfWeekHighlighted: '06',
        todayBtn: true,
        todayHighlight: true
    });
    $('.input-daterange').bizCalendar({
        startDate: '2000-01-01',
        endDate: '2046-12-31'
    });
    $('.inline-date').bizCalendar({
        templates: {
            leftArrow: '<i class="biz-icon">&#xe314;</i>',
            rightArrow: '<i class="biz-icon">&#xe315;</i>'
        },
        theme: 'light-green'
    }).on('changeDate', function(e) {
        $('.selected-date').html(e.format());
    });

Options

参数名称 参数类型 描述
[autoclose] Boolean 选中日期后关闭日历，默认 true
[calendarWeeks] Boolean 显示周数，默认 false
[clearBtn] Boolean 显示清空按钮，默认 false，如果 autoclose 为 true，点击后将关闭日历
[container] String 日历所在容器，默认 'body'
[datesDisabled] Array|String 禁用哪些日期，使用 format 格式的字符串或数组
[daysOfWeekDisabled] Array|String 每周禁用哪几天，格式为 '06'，'0,6' 或 [0,6]
[daysOfWeekHighlighted] Array|String 每周高亮哪几天，格式同 daysOfWeekDisabled
[defaultViewDate] Object 打开日历时的默认视图，格式为 {year: yyyy, month: 0, day: 1}，默认今天
[disableTouchKeyboard] Boolean 移动端禁用键盘，默认 false
[enableOnReadonly] Boolean 可以在 readonly 输入框上启用，默认 true
[endDate] Date|String 最晚结束日期，默认不限制
[format] String 日期格式，默认 'yyyy-mm-dd'
[language] String 语言（zh-CN|en），默认 'zh-CN'
[startDate] Date|String 最早开始日期，默认不限制
[templates] Object 左右箭头模版，默认 {leftArrow: '«', rightArrow: '»'}
[theme] String 主题
[title] String 标题
[todayBtn] Boolean|String 显示切换到今日视图的按钮，默认 false，如为 'linked'，则点击即选中
[todayHighlight] Boolean 高亮今日，默认 false
[weekStart] Number 每周开始日，默认 1

Methods

调用方法时，目标元素需为 input 元素或嵌入式日历的根元素

destroy()

销毁

getDate()

获取日期

hide()

隐藏日历浮层（不适于嵌入式日历）

setDate(Date|String date)

设置日期

setDatesDisabled(Array|String|Boolean dates)

设置禁用哪些日期

setDaysOfWeekDisabled(Array|String|Boolean days)

设置每周禁用哪几天

setDaysOfWeekHighlighted(Array|String|Boolean days)

设置每周高亮哪几天

setEndDate(Date|Boolean date)

设置最晚结束日期

setStartDate(Date|Boolean date)

设置最早开始日期

show()

显示日历浮层（不适于嵌入式日历）


Events

changeDate(e)

日期变更
