Button

HTML


    <button class="ok">OK</button>
    <button class="search">Search</button>
    <button class="disabled">disabled</button>

JavaScript


    $('.ok').bizButton();
    $('.search').bizButton({
        theme: 'light-green',
        icon: 'search'
    });
    $('.disabled').bizButton({
        disabled: true
    });

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[disabled] Boolean 禁用，默认 false
[icon] String 图标名称
[text] String 文字
[size] String 尺寸（small | large），默认 'small'
[theme] String 主题

Methods

destroy()

销毁

disable()

禁用

enable()

激活
