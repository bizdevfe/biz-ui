Textline

HTML


    <div class="textline line1" style="width:300px;height:150px"></div>
    <div class="textline line2"></div>

JavaScript


    $('.line1').bizTextline({
        val: 'a\nb\nc',
        theme: 'deep-orange'
    });
    $('.line2').bizTextline({
        theme: 'deep-orange',
        width: 300,
        height: 150,
        maxLine: 3,
        valArray: ['d', 'e', 'f', 'g', 'h']
    });

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[disabled] Boolean 禁用，默认 false
[height] Number 高度，默认取目标对象高度，最小 52px
[maxLine] Number 最大行数，大于等于 1
[theme] String 主题
[val] String 初始值，字符串形式
[valArray] Array 初始值，数组形式
[width] Number 宽度，默认取目标对象宽度，最小 200px

Methods

destroy()

销毁

disable()

禁用

enable()

激活

length()

获取文本长度（不计回车）

val([String value])

以字符串形式获取/设置内容，超过 maxLine 会被截断

valArray([Array value])

以数组形式获取/设置内容，超过 maxLine 会被截断
