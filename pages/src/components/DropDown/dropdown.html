DropDown







HTML


    <button id="b1">show DropDown</button>
    <div class="dropdown-container">
        <input type="text" placeholder="Username">
        <input type="text" placeholder="Email">
        <input type="text" placeholder="Phone">
        <button id="b2">Sign up</button>
    </div>

JavaScript


    $('input').bizInput();
    $('#b1').bizButton({
        theme: 'blue-gray'
    });
    $('#b2').bizButton({
        theme: 'green'
    });
    $('.dropdown-container').bizDropDown({
        trigger: '#b1',
        customClass: 'register'
    });
    $('#b2').click(function() {
        $('.dropdown-container').bizDropDown('close');
    });

Options

参数名称 参数类型 描述
[alignX] String 水平方对齐方式（left|right），默认 "left"
[alignY] String 垂直方对齐方式（top|bottom），默认 "bottom"
[customClass] String 自定义 class
[onBeforeClose] Function 关闭时的回调，返回 false 则不执行关闭
[onBeforeOpen] Function 打开时的回调，返回 false 则不执行打开
[offsetX] Number 水平偏移，默认 0
[offsetY] Number 垂直偏移，默认 0
trigger String 触发下拉浮层的元素选择器
[zIndex] Number 下拉浮层 z-index，默认使用 trigger 的 z-index

Methods

close()

关闭下拉浮层

destroy()

销毁

open()

打开下拉浮层
