Input

HTML


    <input type="text" class="item1" placeholder="default text">
    <input type="text" class="item2" value="disabled">

JavaScript


    $('input').bizInput().on('enter', function(e, value) {
        bizui.alert(value);
    });
    $('.item2').bizInput('disable');

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[disabled] Boolean 禁用，默认 false
[theme] String 主题

Methods

destroy()

销毁

disable()

禁用

enable()

激活


Events

enter(e, value)

回车
