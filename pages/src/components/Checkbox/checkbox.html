Checkbox



HTML


    <input type="checkbox" name="fruit" value="1" id="apple" title="Apple" checked>
    <input type="checkbox" name="fruit" value="2" id="banana" title="Banana">
    <input type="checkbox" name="fruit" value="3" id="cherry" title="Cherry">
    <button id="b1">check Apple</button>
    <button id="b2">check Banana</button>
    <button id="b3">check Cherry</button>
    <button id="b4">uncheck</button>
    <button id="b5">disable</button>
    <button id="b6">enable</button>
    <button id="b7">get value</button>

JavaScript


    var checkboxGroup = $(':checkbox');
    checkboxGroup.bizCheckbox({
        theme: 'cyan'
    });
    $('button').bizButton({
        theme: 'blue-gray'
    });
    $('#b1').click(function() {
        $('#apple').bizCheckbox('check');
    });
    $('#b2').click(function() {
        $('#banana').bizCheckbox('check');
    });
    $('#b3').click(function() {
        $('#cherry').bizCheckbox('check');
    });
    $('#b4').click(function() {
        checkboxGroup.bizCheckbox('uncheck');
    });
    $('#b5').click(function() {
        checkboxGroup.bizCheckbox('disable');
    });
    $('#b6').click(function() {
        checkboxGroup.bizCheckbox('enable');
    });
    $('#b7').click(function() {
        bizui.alert(checkboxGroup.bizCheckbox('val'));
    });

Options

参数名称 参数类型 描述
[theme] String 主题

Methods

check()

勾选

destroy()

销毁

disable()

禁用

enable()

激活

uncheck()

取消勾选

val()

获取选中的 value 值
