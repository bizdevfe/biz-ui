JavaScript


    $('input').bizInput();
    $('button').bizButton();
    bizui.Tooltip({
        element: 'input',
        theme: 'red'
    });
    bizui.Tooltip({
        element: 'button',
        action: 'hover'
    });
    bizui.Tooltip({
        element: '.protocol',
        preventDefault: true,
        action: 'click'
    });

Options

参数名称 参数类型 描述
[action] String 触发方式（focus | click | hover），默认 focus
[element] String 目标对象选择器，默认 '.error'
[preventDefault] Boolean 阻止默认事件，默认 false
[removeAll] Boolean 移除所有绑定，默认 false
[removeSpecific] Boolean 移除指定绑定，需同时指定 action 和 element，默认 false
[theme] Boolean 主题，默认黑色

 Tips

  • data-tooltip 属性：提示内容，支持简单 HTML 便捷语法。
  • data-tooltip-direction 属性：提示出现位置（right|bottom|left|top），默认 right。
  • Tooltip 的主题默认为黑色，或由参数设置，不受 bizui.theme 影响。
