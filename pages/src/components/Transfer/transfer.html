        theme: 'pink'

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[dataSource] Array 数据
[keyMap] Object 数据字段，默认 {id: 'id', title: 'title', chosen: 'chosen'}
[titles] Array 标题
[noContent] String 无数据提示，不转义
[onChange] Function 移动数据回调，onChange(Object data, Event e)
[theme] String 主题
