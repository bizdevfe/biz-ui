Page








HTML


    <div class="page-nav"></div>
    <button id="b1">prevPage</button>
    <button id="b2">nextPage</button>
    <button id="b3">disable</button>
    <button id="b4">enable</button>
    <input type="text" class="pageNumber">
    <button id="b5">setPageNumber</button>
    <input type="text" class="pageSize">
    <button id="b6">setPageSize</button>
    <input type="text" class="totalNumber">
    <input type="checkbox" title="redaw" class="redaw">
    <button id="b7">setTotalNumber</button>

JavaScript


    var page = $('.page-nav');
    $('button').bizButton({
        theme: 'blue-gray'
    });
    $('input:text').bizInput();
    $('input:checkbox').bizCheckbox();
    page.bizPage({
        pageSize: 50,
        pageNumber: 1,
        theme: 'lime'
    });
    $('#b1').click(function() {
        page.bizPage('prevPage');
    });
    $('#b2').click(function() {
        page.bizPage('nextPage');
    });
    $('#b3').click(function() {
        page.bizPage('disable');
    });
    $('#b4').click(function() {
        page.bizPage('enable');
    });
    $('#b5').click(function() {
        page.bizPage('setPageNumber', $('.pageNumber').val());
    });
    $('#b6').click(function() {
        page.bizPage('setPageSize', $('.pageSize').val());
    });
    $('#b7').click(function() {
        page.bizPage('setTotalNumber', $('.totalNumber').val(), $('.redaw').prop('checked'));
    });

Options

参数名称 参数类型 描述
[customClass] String 自定义 class
[nextText] String 后一页标识
[pageNumber] Number 当前页码，默认1
[pageSize] Number 每页条数，默认1
[prevText] String 前一页标识
[theme] String 主题
[totalNumber] Number 总条数，默认1

Methods

destroy()

销毁

disable()

禁用

enable()

激活

getPageCount()

获取页数

getPageNumber()

获取当前页码

getPageSize()

获取每页条数

nextPage()

向后翻一页

prevPage()

向前翻一页

setPageNumber(Number pageNumber)

设置当前页码

setPageSize(Number pageSize)

设置每页条数，设置后会定位至第一页

setTotalNumber(Number totalNumber[, Boolean redraw])

设置总条数


Events

change(e, pageNumber)

页码变更
